#### <a name="_4p288gjr7ytp"></a>**Overview**
This program allows users to authenticate via Facebook and Twitter, create and manage Facebook business pages, link them with Instagram, and post photos and videos with auto-generated captions.
#### <a name="_od6eiirjajx8"></a>**Features**
-  **Authentication:**
   -  Users can log in using Facebook.
   -  Optional Twitter login for cross-platform posting.

-  **Facebook Business Page Management:**
   -  Authenticated users can create a Facebook business page.
   -  Link a Facebook business page with an Instagram business page.

-  **Posting Content:**
   -  **Photos:**
      -  Enter a prompt to auto-generate an image and caption.
      -  Post the generated photo and caption to the selected Facebook page or Instagram page.

- **Videos:**
   -  Select a video file.
   -  Enter a prompt to auto-generate captions.
   -  Post the video with captions to the selected Facebook page or Instagram page.

-  **Optional Twitter Integration:**
   -  Post the same photos and videos to Twitter if authenticated.
