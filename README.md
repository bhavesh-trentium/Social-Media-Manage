#### <a name="_4p288gjr7ytp"></a>**Overview**
This program allows users to authenticate via Facebook and Twitter, create and manage Facebook business pages, link them with Instagram, and post photos and videos with auto-generated captions. T
#### <a name="_od6eiirjajx8"></a>**Features**
1. **Authentication:**
   1. Users can log in using Facebook.
   1. Optional Twitter login for cross-platform posting.

1. **Facebook Business Page Management:**
   1. Authenticated users can create a Facebook business page.
   1. Link a Facebook business page with an Instagram business page.

1. **Posting Content:**
   1. **Photos:**
      1. Enter a prompt to auto-generate an image and caption.
      1. Post the generated photo and caption to the selected Facebook page or Instagram page.

1. **Videos:**
   1. Select a video file.
   1. Enter a prompt to auto-generate captions.
   1. Post the video with captions to the selected Facebook page or Instagram page.

1. **Optional Twitter Integration:**
   1. Post the same photos and videos to Twitter if authenticated.
